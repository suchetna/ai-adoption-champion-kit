# AI Adoption Champion Kit

Templates and playbooks for building internal AI champions, adoption programs, and team enablement.

## Why this exists

AI adoption rarely works through policy alone. It spreads when trusted people inside teams can model good practice, translate strategy into day-to-day habits, and help others use tools with confidence.

This repository is a working library of practical assets for building internal AI champion programs that support adoption, learning, and responsible use across organizations.

## What this repository will include

- AI champion role outlines
- Team enablement playbooks
- Internal training session templates
- Adoption communication materials
- Champion program planning checklists

## Current status

This repository is being built as a public working library. The first files will focus on champion roles, adoption planning, and simple enablement structures.

## Working principle

Adoption sticks when people inside the organization can explain, model, and support the change.
